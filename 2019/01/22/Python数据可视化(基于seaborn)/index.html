Python数据可视化(基于seaborn)

Posted on 2019-01-22 | In 数据分析

相关依赖

numpy
pandas
seaborn
matplotlib

Seaborn home page https://seaborn.pydata.org/
这些依赖可以通过pip install xxx命令安装
安装完按如下引入

Bar Plot

使用数据集tips
tips=sns.load_dataset(‘tips’)


//使用中位数。柱形显示的为相应数据取中位数后的结果

参数Ci:置信区间

参数capsize:可调横线长度

Saturation：可设置颜色饱和度

Dist Plot

//产生数据


对数据序列化
label_dist=pd.Series(num,name=”variable x”)

vertical=True:可转过来显示

Hist=False:去除阴影
rug=True:显示频谱
sns.distplot(label_dist,rug=True,hist=False)

Box Plot

盒图

引入tips数据集

sns.boxplot(x=’sex’,y=’total_bill’,data=tips)

sns.boxplot(x=’day’,hue=’sex’, y=’total_bill’,data=tips)

sns.boxplot(data=iris,orient=”horizontal”)
orient=”horizontal” 水平显示 ‘h’/’v’

Strip Plot

sns.stripplot(x=tips[‘tip’])

sns.stripplot(x=’day’,y=’total_bill’,data=tips)

添加抖动 jitter=True/数字

Linewidth=1.2 圆点描边大小

加上hue,对sex分类绘制

加上split=True 分离绘制

Marker:形状标记 d菱形,o圆形,s正方形,+
Markers参数可对分类后用不同形状绘制
Size:改变形状大小
Alpha:改变透明度
可以和盒图结合:


Swarmplot()分簇散点图。用法与stripplot差不多。

Pair Grid

iris=sns.load_dataset(‘iris’)

x=sns.pairgird(iris)
x=x.map(plt.scatter)
plt.show()

x=sns.PairGrid(iris)
x=x.map_diag(plt.hist)
绘制变量的概率分布

x=sns.PairGrid(iris)
x=x.map_diag(plt.hist)
x=x.map_offdiag(plt.scatter)
map_diag定义对角线单个属性图，map_offdiag定义非对角线两个属性关系图

对species分类 (使用hue=’species’)

x=x.add_legend()
加上标注

x=x.map_diag(plt.hist,histtype=’step’) #对角线柱形图，只留边框
可用linewidth调整边框宽度

x=sns.PairGrid(iris,vars=[‘petal_length’,’petal_width’])
vars可指定某些变量进行绘图
edgecolor边框颜色
x_vars,y_vars 对x轴与y轴指定某些变量进行绘图
map_upper 对上对角线的图应用某函数
map_lower 对下对角线的图应用某函数

PairGrid中 hue_kws={‘marker’:[‘D’,’S’,’+’]}可分类指定形状

Violin Plot

sns.violinplot(x=tips[‘tip’])

sns.violinplot(x=’size’,y=’total_bill’,data=tips)

Split=true 可合起来

sns.violinplot(x=’size’,y=’total_bill’,data=tips,hue=’sex’,inner=’quartile’)
内部加上四分位数

Scale=’count’指定大小方式
sns.violinplot(x=’day’,y=’total_bill’,data=tips,hue=’sex’,inner=’quartile’,scale=’count’,split=True)

sns.violinplot(x=’day’,y=’total_bill’,data=tips,hue=’sex’,inner=’stick’,scale=’count’,split=True)

bw边界抖动

Clustermap

flights=flights.pivot(‘month’,’year’,’passengers’) #转换成一个二维表 行：月 列：年 内容：乘客数

层化的热力图
sns.clustermap(flights,col_cluster=False)
不显示列的连线

sns.clustermap(flights,col_cluster=False,cmap=’coolwarm’) #调整对比颜色

Linewidth调整小格边框宽度
Figsize={int,int} 调整小格宽高
sns.clustermap(flights,col_cluster=False,cmap=’coolwarm’,standard_scale=1) #标准化

Z_socre=0

Z_socre=方差/Z_val

Heatmap

normal=np.random.rand(12,15) #随机生成12,15的矩阵
sns.heatmap(normal)

Annot=True 在方格上显示数值

min,vmax 调整颜色表示的数值的最大值和最小值


sns.heatmap(flights,fmt=’d’,annot=True)

Center=flight.loc[‘june’,1954] #调整中心位置
Cbar=true/false 是否显示颜色条

FacetGrid

x=sns.FacetGrid(tips,row=’smoker’,col=’time’) #创建了2*2个图 对smoker与time分类
x=x.map(plt.hist,’total_bill’) #对total_bill绘制柱形图

x=x.map(plt.scatter,’total_bill’,’tip’) #绘制散点图

x=x.map(sns.regplot,’total_bill’,’tip’)

x=sns.FacetGrid(tips,col=’time’,hue=’smoker’)
x=(x.map(sns.regplot,’total_bill’,’tip’)).add_legend();

Size:设置图的大小
Aspect=.4

KDEPlot


sns.kdeplot(x_axis,shade=True)
sns.kdeplot(y_axis,shade=True)

sns.kdeplot(x_axis,y_axis)

Shade=True #添加阴影

sns.kdeplot(x_axis,y_axis,cmap=’RdBu’) #调整比较色

N_levels:调整层数
sns.kdeplot(x_axis,y_axis,cmap=’RdBu’,n_levels=18)

iris=sns.load_dataset(“iris”)
setosa=iris.loc[iris.species==’setosa’]
**选取iris中species==’setosa’的item

Joint Plot


sns.jointplot(x=’total_bill’,y=’tip’,data=tips,kind=’reg’) #加上拟合曲线

sns.jointplot(x=’total_bill’,y=’tip’,data=tips,kind=’hex’)

Kind=’kde’
